# BUILD A REST API BACKEND

## REQUIREMENT
- Easily deploy

- No downtime when update

- Monitoring and Logging

- Basic security

- When pushing a new code, automatically deploy

- Only Docker + Docker Compose

## TARGET
- Backend API

- Database (MongoDB)

- Reverse proxy (Nginx)

- CI/CD (local-style)

- Monitoring & Logging

- Security & Hardening

- Zero-downtime deployment

## ARCHITECTURE

- Client --> Reverse proxy --> API --> MongoDB

## STACKS
- Backend: Nodejs (Express)

- Database: MongoDB

- Reverse proxy: Nginx

- Container: Docker

- Orchestration: Docker Compose

- Monitoring: Prometheus + cAdvisor

- Logging: Docker logs

- Security: Trivy

- CI/CD: Github Action
